# ✔ Starlette
- Starlette는 가벼운 ASGI framework/toolkit이다.
- high performance asyncio(비동기) 서비스 이다.
> - 성능이 우수하다.
> - Websocket을 지원한다.
> - GraphQL을 지원한다.
> - CORS, GZip, Static Files, Streaming responses

- python 3.6+
- installation
  - `pip install starlette`
- uvicorn, daphne, or hypercorn을 다운 받아야한다.
  - `pip install uvicorn`

- [참조] https://www.starlette.io/
